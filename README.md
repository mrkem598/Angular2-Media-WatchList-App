# MyWatchList-Angular2App

Media watch List app is a single page Angular2 application!

<img width="1360" alt="screen shot 2017-07-29 at 6 10 27 pm" src="https://user-images.githubusercontent.com/23619819/28748596-abe0a6b8-7489-11e7-967c-d5088869fe0f.png">
    This media watchlist app is a  web application that will allow me 
    to add media lists to the registry focus it down to just
    movies or series, and delete media items from their list.
<img width="1360" alt="screen shot 2017-07-29 at 6 11 23 pm" src="https://user-images.githubusercontent.com/23619819/28748597-abe1d2e0-7489-11e7-9b4e-8a59dcdf464c.png">

    The app will have a single screen experience, and will swap out views for
    lists, and the media item forum. It will be a client side app, with no 
    server requests after the first load.
